# TP3 – Développement du site Web Hollow Knight

## Description du projet
Ce site web est le résultat du TP3 du cours de développement web. L’objectif était de reproduire et améliorer la maquette Figma réalisée pour le TP1 en utilisant HTML, CSS et JavaScript. Le thème s’inspire du jeu Hollow Knight et de l’équipe Team Cherry.

Le site comprend :
- Une page d’accueil avec diaporama d’images et intégration d’une vidéo YouTube.
- Une page de recherche permettant de filtrer des résultats et d’afficher des informations détaillées.
- Une page À propos présentant l’équipe fictive.
- Un formulaire d’inscription amélioré (mise en évidence des champs, vérification, affichage/masquage du mot de passe).
- Une page Contact avec une carte Google Maps affichant une localisation fictive.

## Comment utiliser le site
1. Téléchargez ou clonez le dépôt.  
2. Ouvrez le fichier `index.html` dans votre navigateur pour accéder à la page d’accueil.  
3. Utilisez la navigation pour accéder aux différentes pages.  
Aucune installation particulière n’est nécessaire ; le site fonctionne localement dans n’importe quel navigateur moderne.

## Structure des fichiers
- `index.html` : page d’accueil avec carrousel et vidéo.  
- `recherche.html` : page de recherche avec liste filtrable.  
- `about.html` : présentation de l’équipe et du projet.  
- `inscription.html` : formulaire d’inscription interactif.  
- `contact.html` : formulaire de contact et carte Google Maps.  
- `css/styles.css` : feuilles de style globales.  
- `js/script.js` : fonctions JavaScript (carrousel, recherche, validation du formulaire).  
- `images/` : toutes les images du site.  
- `rapport_developpement.md` : rapport de développement (version remise).
